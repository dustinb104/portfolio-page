The Run-Down House

This project was created as part of a class assignment in my full stack web applications class at the University of New Orleans. In this project I was to create an interactive story using only HTML, CSS, and Bootstrap. The idea of this project was to learn about the basics of these technologies and about how to structure and deploy them to create a simple website. All of the ideas used within this project are all basic because I am still in the process of learning about them. In the future as I get more advanced I plan on making changes to the project to improve upon the ideas to make it more appealing to the user.
